Action Dispatch AssumeSSL

When proxying through a load balancer that terminates SSL, the forwarded request will appear as though it‚Äôs HTTP instead of HTTPS to the application. This makes redirects and cookie security target HTTP instead of HTTPS. This middleware makes the server assume that the proxy already terminated SSL, and that the request really is HTTPS.
