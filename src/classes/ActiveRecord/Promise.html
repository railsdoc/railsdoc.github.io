value()

Returns the query result. If the query wasn‚Äôt completed yet, accessing value will block until the query completes. If the query failed, value will raise the corresponding error.

def value
  return @value if defined? @value

  result = @future_result.result
  @value = if @block
    @block.call(result)
  else
    result
  end
end
