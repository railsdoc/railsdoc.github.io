inheritance_column

The name of the table column which stores the class name on single-table inheritance situations.

The default inheritance column name is type, which means it‚Äôs a reserved word inside Active Record. To be able to use single-table inheritance with another column name, or to use the column type in your own model for something else, you can set inheritance_column:

If you wish to disable single-table inheritance altogether you can set inheritance_column to nil

self.inheritance_column = nil
