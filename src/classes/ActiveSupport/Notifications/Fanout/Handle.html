A Handle is used to record the start and finish time of event.

Both start and finish must each be called exactly once.

Where possible, it‚Äôs best to use the block form: ActiveSupport::Notifications.instrument. Handle is a low-level API intended for cases where the block form can‚Äôt be used.

ensure

end
