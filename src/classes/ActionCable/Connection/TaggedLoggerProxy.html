Action Cable Connection TaggedLoggerProxy

Allows the use of per-connection tags against the server logger. This wouldnâ€™t work using the traditional ActiveSupport::TaggedLogging enhanced Rails.logger, as that logger will reset the tags between requests. The connection is long-lived, so it needs its own set of tags for its independent duration.
