view_context()

An instance of a view class. The default view class is ActionView::Base.

The view class must have the following methods:

  • View.new(lookup_context, assigns, controller) ‚Äî Create a new ActionView instance for a controller and we can also pass the arguments.

  • View#render(option) ‚Äî Returns String with the rendered template.

Override this method in a module to change the default behavior.
